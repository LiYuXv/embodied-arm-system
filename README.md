# 基于自然语言交互的机械臂具身操作系统设计与实现

## 1. 项目简介

本项目为本科毕业设计项目，目标是设计并实现一个基于自然语言交互的机械臂具身操作系统。系统通过自然语言指令理解、视觉感知、任务规划与机械臂控制等模块，实现用户输入语言指令后，机械臂能够在仿真环境及真实平台中完成简单具身操作任务。

## 2. 预期系统流程

用户自然语言输入
→ 语言模型解析任务意图
→ 视觉模块识别目标物体与目标区域
→ 任务规划模块生成机械臂操作步骤
→ 机械臂运动规划与控制
→ 仿真验证
→ 真实机械臂平台部署与实验验证

## 3. 初步任务场景

第一阶段以简单桌面操作任务为目标，例如：

* “把红色方块移动到蓝色区域”
* “抓取桌面上的目标物体”
* “将指定物体放置到指定位置”

## 5. 当前阶段目标

* 完成项目仓库搭建；
* 整理毕业设计任务书和进度计划；
* 明确系统总体技术路线；
* 调研具身智能、VLA、机械臂控制与仿真相关文献；
* 确认机械臂平台软硬件环境和开发接口。

## 6. 技术路线初稿

本项目计划采用“自然语言理解 + 视觉感知 + 机械臂控制”的模块化方案。前期优先打通从语言指令到机械臂仿真执行的最小闭环，后续根据进展引入 VLA 模型或更复杂的任务规划方法，并尝试完成从仿真环境到真实机械臂平台的迁移。
